## How to Use This Ebook

**Night before the interview:** Read Chapter 16 (Cheat Sheet) cover to cover. It summarizes everything.

**Study session (2-3 hours):** Read Part 2 (Chapters 8-15) — this is the UI Architect core.

**Deep preparation (full day):** Read everything front to back. Each chapter is self-contained.

**During the interview:** Use the 5-step framework from Chapter 1. Always start with requirements.
